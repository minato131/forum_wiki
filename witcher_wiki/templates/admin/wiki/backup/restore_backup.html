Восстановление базы данных

Важная информация!

Вы собираетесь восстановить базу данных из резервной копии. Это действие:

  • Создаст резервную копию текущей базы данных (автоматически)
  • Удалит все текущие данные из базы
  • Восстановит данные из выбранного файла
  • Работает только с файлами бэкапа созданными в этой системе

Это действие нельзя отменить!

Поддерживаемые форматы

Формат файла

  • ZIP архив с JSON данными
  • Созданный через "Создать бэкап"
  • Содержит backup_data.json

База данных

  • Только SQLite
  • Полная структура таблиц
  • Все типы данных

Безопасность

  • Проверка формата
  • Автобэкап текущей БД
  • Валидация данных
